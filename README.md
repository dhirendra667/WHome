Whome is a modern, full-stack web application designed to connect hosts and travelers for property rentals. 
This platform uses the MERN stack (MongoDB, Express, React, Node.js) with Redux for state management and features secure authentication, property listing management, and a dynamic user experience.

Key Features Implemented:
Host Dashboard: Dynamic visibility for hosts based on property ownership.

Redux Toolkit: Efficient and predictable state management (with redux-persist for caching).

Secure File Upload: Image uploads handled via Cloudinary.

Authentication: User login/registration with secure, cookie-based authentication.
